# LLM Red Teaming & Safety Testing

## Overview
This project evaluates LLM robustness against adversarial prompts including:

- Prompt injection
- Jailbreak attempts
- Instruction override attacks

## Objective
Identify vulnerabilities and assess model safety risks.

## Methodology
Each test includes:
- Attack prompt
- Model response
- Vulnerability classification
- Risk level
- Analysis

## Importance
AI safety is critical in production systems. This project demonstrates real-world testing techniques used in LLM security evaluation.
